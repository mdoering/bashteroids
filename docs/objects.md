# Add more objects and options to the game

## personalize users
Allow users to enter a short name (8 chars max) and use that for the scores & highscore.
Entering a name should be done via the keyboard.
The name will be requested when adding the controller to the game via the A button.
In subsequent runs the name will stay, but if a user hits the A button again the name can be changed.
Show the name underneath the squares.

## Allow single user games
When just a single controller is added to the game allow to start the game in solo mode.
The user then has to survive the enemies and the game stops when there the only user dies.


## increase difficulty over time
The longer a game takes, the harder it should become to survive.
The time elapsed should determine how many objects are created and how fast these initially are.
Add more and speedier enemies for every minute of the game, slowly increasing the amount and speed.


## updates to existing objects

### laser instead of gun bullets
Show a real short line for a laser shot instead of a round ball for bullets.

### asteroid shape
Asteroids should have a dim grea circle in the inside of their hull.


## more bonus objects
Add objects to the game that a ship can run into to pick it up and gain new features.
These objects run through the screen only once like the bullets and do not reappear on the other side.
The follow a straight trajectory with a fixed speed (varying between items of course).

### shield
A defensive shield that once picked up by a ship will add a ring around it that absorbes one hit/collision, so that the ship does not crash.
The shield will be gone after one hit.

### twin laser
A second laser barrel that improves the shooting ability of a ship. It can fire 1.5 as often as the regular gun and bullets leave the ship alternating slightly from the left and right of the ship, not the exact middle.

## enemy objects
Add some more evil items like the UFO.

### space mine
A sea mine / bomb that explodes after a few seconds and kills anything in a ship long radius around it. Large explosion animation.
The bomb flashes slowly in the beginning and gets faster as it gets close to its explosion.
Scores 5

### alien monster
Moves like an UFO, but has short range lasers
Scores 10


## controls
Accept the X button (current fire button) to start the game.
Allow the right LR or L2 button to be used for shooting.
